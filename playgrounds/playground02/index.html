Playground 02

Agora vamos manipular os elementos de uma lista

Lista

A lista a seguir se chama minhalista.

  • Item 0
  • Item 1
  • Item 2
  • Item 3
  • Item 4

Tarefas

Tarefa 1

Mostre quantos elementos existem na lista.

minhalista.childElementCount

Tarefa 2

Mude o texto do Item 3 da lista

minhalista.children[3].textContent = 'Oi'

Tarefa 3

Faça o último item da lista ter o mesmo texto do primeiro item

minhalista.children[4].textContent = minhalista.children[0].textContent

Tarefa 4

Remova o primeiro item da lista.

minhalista.removeChild(minhalista.children[0])

Tarefa 5

Risque um dos itens da lista

minhalista.children[2].style.textDecoration = 'line-through'
