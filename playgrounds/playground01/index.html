Playground 01

Vamos começar aprendendo a encontrar e manipular elementos dentro de uma página. O mais simples será um texto

Blocos

Eu me chamo caixa.

Tarefas

Tarefa 1

Mude o texto do bloco acima para uma frase qualquer usando apenas comandos no terminal.

caixa.innerHTML = 'Isso é um texto'

Tarefa 2

Mude o cor da borda do bloco acima para vermelho.

caixa.style.borderColor = 'red'

Tarefa 3

Aumente o tamanho da letra do bloco acima.

caixa.style.fontSize = '3em'

Tarefa 4

Vire o bloco de cabeça para baixo.

caixa.style.rotate = '180deg'

Tarefa 5

Torne o bloco acima invisível

caixa.style.visibility = 'hidden'
